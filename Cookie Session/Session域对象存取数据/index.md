# 怎样向Session中存储数据（session也是一个域对象）什么是域对象？？？

Session也是存储数据的区域对象，所以session对象也具有如下三个方法：  
session.setAttribute(String name,Object obj);  
session.getAttribute(String name);  
session.removeAttribute(String name);
